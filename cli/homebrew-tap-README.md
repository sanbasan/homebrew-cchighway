# CCHighway Homebrew Tap

This is the official Homebrew tap for CCHighway (Claude Code Highway).

## Installation

```bash
# Add the tap
brew tap sanbasan/cchighway

# Install CCHighway
brew install cchighway

# Verify installation
cchighway --version
```

## Usage

```bash
# Start the server
cchighway start

# Check status
cchighway status

# Stop the server
cchighway stop

# Interactive setup
cchighway init
```

## Requirements

- macOS 10.15+
- Node.js 18+ (automatically installed as dependency)
- Claude Code CLI tool installed separately
